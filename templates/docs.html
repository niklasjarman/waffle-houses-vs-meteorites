API Documentation

For any of this API, you can put the HTTP GET parameter ?response=xml at the end of the URL to retrieve the XML version.

Meteorite Landings API

This API provides information about meteorite landings.

  • GET /api/meteorite_landings

    • Retrieves a list of all meteorite landings.

  • GET /api/meteorite_landings/name/<name>

    • Retrieves information about a meteorite landing by its name.

  • GET /api/meteorite_landings/type/<recclass>

    • Retrieves information about meteorite landings by their classification type.

  • GET /api/meteorite_landings/min_mass/<min_mass>/max_mass/<max_mass>

    • Retrieves meteorite landings within a specified mass range.

  • GET /api/meteorite_landings/type/<recclass>/year/<year>

    • Retrieves meteorite landings of a specific classification type within a given year.

Waffle Houses API

This API provides information about Waffle House locations.

  • GET /api/waffle_houses

    • Retrieves a list of all Waffle House locations.

  • GET /api/waffle_houses/name/<name>

    • Retrieves a Waffle House location based on it's name.

  • GET /api/waffle_houses/state/<state>

    • Retrieves Waffle House locations in a specific state.

  • GET /api/waffle_houses/state/<state>/zipcode/<zipcode>

    • Retrieves Waffle House locations in a specific state with a given ZIP code.

  • GET /api/waffle_houses/count_by_state

    • Retrieves the count of Waffle House locations in each state.

  • GET /api/waffle_houses/count_by_state/state/<state>

    • Retrieves the count of Waffle House locations in a specific state.

Average Distance API

This API calculates the average distance between meteorite landings and Waffle House locations.

  • GET /api/average_distance

    • Calculates the overall average distance between meteorite landings and Waffle House locations.

  • GET /api/average_distance/year/<year>

    • Calculates the average distance between meteorite landings and Waffle House locations for a specific year.

  • GET /api/average_distance/year/<year>/waffle_house_state/<state_initials>

    • Calculates the average distance between meteorite landings and Waffle House locations for a specific year in a particular state (e.g., SC).

Closest Pairs API

This API finds the closest pairs of meteorite landings and Waffle House locations.

  • GET /api/closest_pairs?limit=num

    • Retrieves the closest pairs of meteorite landings and Waffle House locations, limited by the specified number.

  • GET /api/closest_pairs/distance/<distance> (float)

    • Retrieves the closest pairs of meteorite landings and Waffle House locations within the given distance threshold.

Closest Meteorites API

This API finds meteorite landings closest to a specific location.

  • GET /api/closest_meteorites/waffle_house_name/<waffle_house_name>

    • Retrieves meteorite landings closest to a specific Waffle House location by its name.

  • GET /api/closest_meteorites_to_me/max_distance/<distance>

    • Retrieves meteorite landings closest to a given location within the specified distance.

Closest Waffle House to Meteorites API

This API finds the closest Waffle House to a specific meteorite landing.

  • GET /api/closest_waffle_house_to_meteorites

    • Retrieves the closest Waffle House to each meteorite landing.

  • GET /api/closest_waffle_house_to_meteorites/meteorite_name/<name>

    • Retrieves the closest Waffle House to a specific meteorite landing by its name.

Closest Waffle Houses to Me API

This API finds the closest Waffle Houses to a given location.

  • GET /api/closest_waffle_houses_to_me/max_distance/<distance>

    • Retrieves the closest Waffle Houses to a given location within the specified distance.

Closest Meteorites to Me API

This API finds the closest meteorite landings to a given location.

  • GET /api/closest_meteorites_to_me/max_distance/<distance>

    • Retrieves the closest meteorite landings to a given location within the specified distance.

Nearby Locations API

This API finds nearby locations based on latitude and longitude coordinates.

  • GET /api/nearby_locations/latitude/<latitude>/longitude/<longitude>

    • Retrieves nearby locations based on the provided latitude and longitude coordinates.
